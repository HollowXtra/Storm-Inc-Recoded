## STORM_INC - Tropical Cyclone Simulator
A vibe project using frontier LLMs, NOT CAREFULLY REVIEWED, so there might be unexpected bugs.

## Features
- **Tropical Cyclone Simulator** (TCM Interface) - Realistic storm simulation with:
  - Death Counter & Damage Counter
  - Realistic movement physics (beta drift, steering flow, land interaction)
  - Weather Warnings & Watches system
  - Doppler Radar & Satellite View
  - Forecast Track & Wind Radii
- **AI-IR Tropical Cyclone Intensity Identifier** - Uses ONNX model for intensity estimation
- **Wind Data Grabber** - Fetches real wind data
